Jacques Lycanthantropy

This is a task that involves keeping a log of Jacques a country guy that doesn't know why he turns to a weresquirrel.

The following are the aims of this task

  • To structure the logs of the event in a suitable manner
  • To use the phi correlation coefficient to determine the particular event that determines his Lycanthantropy
  • To make sure that the logs is just for two weeks
  • To make sure that it is pizza that changes him to squirell
